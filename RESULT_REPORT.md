# 方块世界 3.0 优化报告

这次更新目标是把原型小游戏升级成更接近体素沙盒的版本，重点放在地图规模、交互、视觉氛围、物品栏、天气、动植物和移动端可玩性。

## 已完成的 48 项优化

1. 新增 `src/v3.js`，作为 3.0 主游戏引擎。
2. 新增 `src/v3.css`，重做移动端 HUD、物品栏、快捷栏和按钮样式。
3. `index.html` 切换到 3.0 入口。
4. 增加完整物品栏入口：顶部 🎒。
5. 增加物品栏弹窗。
6. 增加物品分类标签：全部、自然、植物、矿物、建筑。
7. 增加物品图标样式，不再只是纯色方块。
8. 增加底部快捷栏。
9. 增加数字键 1-9 切换快捷栏方块。
10. 增加 30+ 种方块和物品定义。
11. 增加草方块、泥土、石头、沙子、砂砾、黏土、雪、冰、水。
12. 增加原木、白桦木、树叶、松叶、仙人掌、芦苇、草丛、红花、黄花、蘑菇。
13. 增加煤矿、铁矿、金矿、晶簇、玄武岩、黑曜石。
14. 增加木板、砖块、玻璃、灯块、白墙、深色石、蓝砖、红砖。

19. 区分实体方块和透明/植物方块渲染。

22. 增加山丘、细节和山脊变化。
23. 增加河流生成。
24. 增加多条河流合流的地形逻辑。

26. 增加森林、平原、沙漠、湿地、雪地、冰河等生物群系。

## 操作入口

- 🎒：物品栏
- 💾：保存世界
- 🎲：随机刷新新世界
- ⏱️：调整昼夜速度
- 🌦️：切换天气
- 👁️：调整视野区块距离
- 🏠：回到出生点

## 仍可继续做的方向

- 加入真正的背包数量、采集掉落和合成台。
- 加入生命值、饥饿值和伤害。
- 加入更复杂的洞穴和地下结构。
- 加入建筑蓝图、门、梯子、火把等交互方块。
- 加入更细的移动碰撞，而不是目前的高度采样碰撞。
- 加入音效文件和背景音乐，而不是当前的简单 WebAudio 方波反馈。
